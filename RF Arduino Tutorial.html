RF Transmitter and Receiver Arduino Tutorial

Using two Arduinos, create sketches to transmit a 12-character string containing one number with a leading ‘X’ and padded with ‘Z’. The receiver should display the numbers and indicate whether the number is even or odd.

Wiring schematic for the RF Transmitter:

The Arduino code for the RF Transmitter 


                #include RH_ASK.h
                #include SPI.h // Not actually used but needed to compile
                
                RH_ASK driver;
                                
                void setup()
                {
                    Serial.begin(9600);	  // Debugging only
                    if (!driver.init())
                    Serial.println("init failed");
                }
                                
                void loop()
                {
                    const char *msg = "X13ZZZZZZZZZ";
                    driver.send((uint8_t *)msg, strlen(msg));
                    driver.waitPacketSent();
                    delay(1000);
                }
